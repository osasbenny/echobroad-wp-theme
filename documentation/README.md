# EchoBroad WordPress Theme

A modern, fully functional WordPress theme replicating the EchoBroad Agency website design.

## Features

- **Responsive Design**: Mobile-friendly and works on all devices
- **Custom Post Types**: Services, Courses, Products, and Testimonials
- **Homepage Sections**: Hero, Features, Services, Brands, Products, Courses, Testimonials, Blog, FAQ, and CTA
- **Custom Meta Boxes**: Price, level, and other details for courses and products
- **Navigation Menus**: Primary and footer menus
- **Customizer Options**: Contact information and social media links
- **FAQ Accordion**: Interactive FAQ section
- **Smooth Animations**: Scroll animations and hover effects
- **Contact Form**: Built-in contact form functionality

## Installation

1. Download the theme files
2. Upload to `/wp-content/themes/` directory
3. Activate the theme in WordPress admin
4. Configure theme settings in Appearance > Customize

## Theme Setup

### 1. Configure Logo
- Go to Appearance > Customize > Site Identity
- Upload your logo

### 2. Set Up Menus
- Go to Appearance > Menus
- Create a menu with: Home, About Us, Our Services, E-Store, Courses, Blog, Contact
- Assign to "Primary Menu" location

### 3. Configure Contact Information
- Go to Appearance > Customize > Contact Information
- Add email, phone, and address

### 4. Add Social Media Links
- Go to Appearance > Customize > Social Media Links
- Add your social media URLs

### 5. Create Content

#### Services
- Go to Services > Add New
- Add title, description, and featured image
- Publish

#### Courses
- Go to Courses > Add New
- Add title, description, and course details
- Set price and level in the Course Details meta box
- Publish

#### Products
- Go to Products > Add New
- Add title, description, and product details
- Set price and icon in the Product Details meta box
- Publish

#### Testimonials
- Go to Testimonials > Add New
- Add testimonial text as content
- Fill in author name, position, and rating
- Publish

### 6. Create Pages

Create the following pages:
- Home (set as homepage in Settings > Reading)
- About Us
- Contact (use "Contact Page" template)
- Blog (set as posts page in Settings > Reading)
